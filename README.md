| 节数 |         课程         |                             说明                             |                讲师                |
| :--: | :------------------: | :----------------------------------------------------------: | :--------------------------------: |

|  5   |     机器学习基础     |  机器学习的基本概念和方法，如标签，指标，特征，常用模型等。  |               刘锦荣               |
